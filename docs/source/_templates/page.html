Open-source · Python · HPC-ready

Self-consistent light–matter simulations, from a laptop to an HPC cluster.

MaxwellLink couples electromagnetic solvers with quantum and classical molecular drivers through a unified Python interface and a socket-based communication layer.

Install Documentation Tutorials

Designed for heterogeneous light–matter science

Mix and match EM solvers and molecular drivers at will. Each component communicates through a common socket protocol, so different modules can run on separate processes or computing nodes without modifying the simulation logic.

EM solvers

Swap the electromagnetic backend without rewriting your simulation — from full 3-D FDTD to rapid single-mode prototyping.

Meep FDTD Single-mode cavity Prescribed field

Molecular drivers

Combine drivers at different levels of theory within a single EM run, from effective open quantum systems to first-principles electron dynamics and classical MD.

Get started in minutes

MaxwellLink supports the standard Python interface for easy usage.

  1. Install the package: pip install maxwelllink
  2. Select an EM solver and one or more molecular drivers.
  3. Run a tutorial and build from there.
